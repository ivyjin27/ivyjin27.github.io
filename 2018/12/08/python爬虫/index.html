python爬虫

发表于 2018-12-08 | 分类于 python

之前断断续续的学习了一些python爬虫相关的知识，但是一直没有好好总结一下，代码也仅仅局限于ctrl+c和ctrl+v。于是给自己一个任务：从拉勾网爬数据分析师JD，了解一下目前数据分析师的工资待遇、核心技能需求等。
从这个课题立项到正式整理文章还是经过了几天时间，这几天也慢慢get到了爬虫的技能。python爬虫的包有requests/urllib/scrapy，数据提取工具有BeautifulSoup和json。其中BeautifulSoup类似SQL，json接近于DataFrame。但是具体的差异和优劣，目前”技能库“还有所欠缺，仅仅记录下用requests爬取拉勾网职位列表和JD明细信息，并通过json和BeautifulSoup清洗数据的思路。

基础知识点梳理

requests

参数列表：官方文档有描写很多参数，这里就简单整理几个常用的

参数 含义
method 指定request对象的方法
url 指定Request 对象的 URL
params（可选) 指定 Request对象的查询字符串，可以是字典（Dict）或者字节数组（Bytes）
data（可选） 指定 Request 对象的 body 内容，可以是字典（Dict）、包含元组的列表 [(key, value)]、字节数组（Bytes）或者类文件对象
json（可选） 指定 Request 对象的 body 内容，是以 JSON 格式的形式
headers（可选） 指定 Request 对象的 HTTP 头部内容，是以字典（Dict）的形式
